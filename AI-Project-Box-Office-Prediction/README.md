Created in collaboration with with Kabhilesh Giri.

A machine learning framework to predict movie box office revenue ranges during the development stage. Engineered features from structured and unstructured data (one-hot encoding, TF-IDF) and implemented multiple models—including Linear Regression, Random Forest, XGBoost, SVM, LightGBM, and CatBoost—to capture nonlinear relationships. Achieved 0.6 R-Squared accuracy, better than previous research attempts.

**FinalNotebook.ipynb:**

Written in collaboration with Kabhilesh Giri. All stages of our pipeline from preprocessing the data to graphing the training and test results. The file uses linear regression by default but this can be swapped out with any other model. 

**Moives-Dataset.csv:**

The full unprocessed dataset read in by FinalNotebook.ipynb

**text_processing_clustering_experiment.ipynb:**

Conceived and written on my own. The full experiment where I attempted to handle text data through clustering. See the project report for a more detailed breakdown. 

**Movies_Dataset_Original.csv:**

The full unprocessed dataset read in by text_processing_clustering_experiment.ipynb. Slightly different to Moives-Dataset.csv.
